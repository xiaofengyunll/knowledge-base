# 一个大型局域网面临的问题
 1. 局域网内任何一个节点。发出的广播帧，都会被广播至所有节点。可能出现“广播风暴”。
 2. 不安全，安全局域网内可能有一些敏感节点。

# VLAN
- 将一个大型局域网分割成若干个较小的VLAN，每个VLAN是一个广播域。
- 需要使用支持VLAN功能的以太网交换机来实现。
- 每个VLAN对应一个 VID。

**VLAN划分方式**：
1. 基于接口
2. 基于 MAC 地址
3. 基于 IP 地址，这种方式可以让VLAN范围跨越路由器，让多个局域网的主机组成一个VLAN（需要网络层功能支挂）

注：
- 如果帧的传输跨越交换机，需指明VID
- 主机与交换机之间，传输标准以太网帧
- 交换机与交换机之间，传输802.1Q帧

**802.1Q帧的结构**
在标准以太网帧的源地址之后，插入VLAN标签，得到802.1Q帧。
